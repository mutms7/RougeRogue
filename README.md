# RougeRogue

• Developing a roguelike dungeon crawler in C# using the RogueSharp library for core game systems

• Implementing randomized dungeon generation with hallways connecting rooms on map

• Implemented field-of-view and line-of-sight systems for monster awareness and interactability

• Creating character systems for player stats, monster behaviors, and health tracking

Tech Stack: 

C#, RogueSharp, RLNET
